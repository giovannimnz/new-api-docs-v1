## NewAPI Integration Method

AstrBot supports integrating NewAPI as a model provider, allowing users to access and utilize various AI model services through NewAPI.

### Configuration Steps

#### Obtain NewAPI API Key

After registering and logging into NewAPI, click "Console" in the top navigation bar, then click "Token Management", and then click the "Add Token" button to create a new API Key. Select appropriate permissions, then click "Create".

![create-api-key](/assets/astrbot/image.png)

After successful creation, click the copy key button to copy the generated API Key.

![copy-api-key](/assets/astrbot/image-1.png)

#### Configure NewAPI Service Provider in AstrBot

Open the AstrBot management panel, navigate to the "Model Providers" page, then click the "Add New Model Provider" button.

NewAPI perfectly supports OpenAI Chat Completion and Responses APIs. Click "OpenAI" to enter the OpenAI provider's configuration page.

In the pop-up dialog box, set the API Base URL to NewAPI's API address. If you have deployed NewAPI locally, fill in the local address, for example, `http://localhost:3000/v1`. If you are using NewAPI services provided by a third-party service provider, fill in the corresponding URL address, for example, `https://api.example.com/v1`.

Then, enter the API Key into the "API Key" field and click the "Save" button.

![astrbot-provider-config](/assets/astrbot/image-2.png)

Then click Save to complete the NewAPI provider configuration.

#### Apply Service Provider

Go to the "Profile" page, find the Models section, change the "Default Chat Model" to the newly created NewAPI provider, and click the "Save" button.

![apply](/assets/astrbot/image-3.png)

At this point, you have successfully configured NewAPI as AstrBot's model provider. Now, you can access and use various AI model services provided by NewAPI through AstrBot.
