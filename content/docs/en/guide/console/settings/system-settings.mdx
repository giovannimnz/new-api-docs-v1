title: System Settings
---

Here you can configure NewAPI worker, mail server, and login and registration related settings.

![系统设置1](/assets/guide/system-setting-1.png)

![系统设置2](/assets/guide/system-setting-2.png)

When OIDC configuration is enabled, please be sure to check "Allow new user registration", otherwise new users logging in via OIDC will not be able to create users normally. Also check "Allow login via OIDC". Check other configurations as needed.
