## Visão Geral

**Skills** é um protocolo leve de extensão que permite que editores e agentes de IA — como Claude Code, Codex, OpenClaw, Cursor e Windsurf — interajam com a New API diretamente do ambiente de coding, sem depender de navegador ou painel admin.

A Skill `newapi` já cobre consulta de modelos, gestão de tokens, verificação de saldo e perguntas sobre uso da New API.

## Editores compatíveis

| Editor / Assistente | Tipo | Notas |
|---------------------|------|-------|
| **Claude Code** | Assistente de programação no terminal | CLI oficial da Anthropic |
| **OpenClaw** | Plataforma self-hosted de agentes | Suporta fluxos de comando com `/newapi` |
| **Cursor** | Editor nativo de IA | Baseado em VS Code |
| **Windsurf** | Editor de código com IA | Mantido pela Codeium |
| **Cline** | Extensão de IA para VS Code | Agente open-source |
| **Codex CLI** | Assistente de programação no terminal | CLI oficial da OpenAI |

> Qualquer ferramenta compatível com o protocolo de Skills pode usar estas integrações.
