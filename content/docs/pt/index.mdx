## Por que Atius AI Router?

O **Atius AI Router** é um fork orientado a produção do `QuantumNous/new-api`, com foco em estabilidade operacional, documentação organizada e compatibilidade prática com fluxos OpenAI / Anthropic.

- API única para múltiplos providers
- Compatibilidade com SDKs e ferramentas que já falam OpenAI
- Documentação local em PT-BR para reduzir atrito operacional
- Base open-source auditável
