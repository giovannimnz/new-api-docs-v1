<Callout type="info">
  🌊
  FluentReadは、母国語のような読書体験を提供することを目指したオープンソースのブラウザ翻訳プラグインです。
</Callout>

- プロジェクトアドレス：[https://github.com/Bistutu/FluentRead](https://github.com/Bistutu/FluentRead)

## 🌟 コア機能

### スマート翻訳エンジン

- **マルチエンジン対応**：20種類以上の翻訳エンジンをサポート
- **従来の翻訳**：Microsoft翻訳、Google翻訳、DeepL翻訳など
- **AI大規模モデル**：OpenAI、DeepSeek、Kimi、Ollamaなど
- **カスタムエンジン**：カスタム翻訳サービス設定をサポート

### 没入型読書体験

- **バイリンガル対照**：原文と翻訳文が並列表示され、より読みやすくなります
- **単語選択翻訳**：任意のテキストを選択すると、即座に翻訳結果が得られます
- **ワンクリックコピー**：翻訳文を素早くコピーし、読書効率を向上させます
- **全文翻訳**：フローティングボールでウェブページ全体をワンクリックで翻訳、ページのリフレッシュは不要です

### プライバシーとカスタマイズ

- **プライバシー保護**：すべてのデータはローカルに保存され、コードはオープンソースで透明です
- **高度なカスタマイズ**：豊富なカスタムオプションで、さまざまなシナリオのニーズに対応します
- **完全無料**：オープンソースで無料、非営利プロジェクト

## 🚀 設定方法

### New API コンソールから設定をインポート（推奨）

ブラウザにFluentReadプラグインがインストールされている場合、New API コンソール->トークン管理ページを開くと、FluentReadを追加するためのプロンプトが表示されます。

![添加提示](/assets/fluentread/hint.png)

モデルを選択した後、FluentReadにワンクリックで入力をクリックすると、確認ウィンドウが表示され、対応する情報が正しいか確認してください。

![确认](/assets/fluentread/confirm.png)

インポートを確認すると、FluentReadのNew API設定が有効になります。

![配置结果](/assets/fluentread/fluentread.png)

### FluentReadで手動で設定を入力する

![手动配置](/assets/fluentread/configuration.png)

| 設定項目     | 内容                       |
| ---------- | -------------------------- |
| 翻訳サービス   | NewAPI                     |
| アクセストークン   | NewAPI キー                |
| NewAPIインターフェース | NewAPIデプロイアドレス（/v1なし）  |
| モデル       | リストから選択、またはカスタムモデル |
| カスタムモデル | モデル名                   |
